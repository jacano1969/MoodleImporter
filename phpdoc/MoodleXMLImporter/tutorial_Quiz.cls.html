Quiz Class

HTML Test Import Specification

  1. The order in which questions are provided to the system will be preserved in the ?Review Question? view and in the output XML file.
  2. Each question in the content to be imported (whether copied and pasted or uploaded in a document) will conform to the following specifications:
  3. Each question will begin with: 
    <h2>Question name OR N/A</h2>
  4. The question text will immediately follow the closing h2 tag.
  5. The question text may contain any legitimate HTML tag, including, but not limited to, tables, paragraphs, and unordered and ordered lists.
  6. Essay questions must end with an empty _____ tag
  7. Multiple-choice and true/false questions must end with either an "ol" or "ul" tag.
  8. Multiple-choice questions that end with an "ol" tag will disable shuffling of its answer choices
  9. Multiple-choice questions that end with an "ul" tag will enable shuffling of its answer choices
  10. Each possible answer choice to a multiple-choice question must appear within "li" tags inside the "ol" or "ul" set of tags.
  11. Correct answers for multiple-choice questions will be embedded within a 
    <strong></strong>
    set of tags.
  12. Matching questions must end with a "dl" tag, with each term and its correct definition appearing in "dt" and "dd" tags, respectively.
  13. True/false questions must enclose either a T (or true) or an F (or false) within an "li" set of tags to indicate the correct answer.
  14. Only the correct answer is specified for true/false questions.
  15. Questions will be given a name that is consistent with the following naming convention: TTnnn ? [question name] Where, TT is the question type (MC = Multiple choice, TF = true/false, ES = Essay, MT = Matching), nnn is the question number formatted to three digits (e.g., 001 for question #1), and [question name] is the contents of the h2 tag OR the first five words of question text, if the h2 tag contains ?N/A?.
  16. Questions will be numbered sequentially from the beginning of the file to the end, in the order in which the questions are submitted.
